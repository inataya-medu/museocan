# Blog-template

This is a blog template with SvelteKit for Ouvroir’s lab members.

Visit [ouvroir/blog-template](https://github.com/ouvroir/blog-template) for more information about this project.

This template uses SvelteKit. See the [Svelte Kit documentation](https://svelte.dev/docs/kit).

## Project creation

Fork this repository and clone it in the desired directory.

Then

```bash
git clone https://github.com/my_git_account/blog-template project_name
cd project-name
```

## Developing

Once you’ve created a project and installed dependencies with `npm install`, start a development server :

```sh
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

To create a production version of your app:

```sh
npm run build
```

You can preview the production build with `npm run preview`.

> To deploy your app, you may need to install an [adapter](https://svelte.dev/docs/kit/adapters) for your target environment.
